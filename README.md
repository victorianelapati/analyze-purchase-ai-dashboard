# Conversational AI Dashboard 📊🤖

## 🚀 Overview

This project is a Conversational AI-powered dashboard that allows users to analyze business data using natural language queries. Users can upload CSV datasets and ask questions like "Show gender distribution" or "Compare monthly income", and the system generates insights instantly.

---

## ✨ Features

* 📁 Upload CSV datasets
* 💬 Ask questions in natural language
* 📊 Automatic chart & insight generation
* 🌐 User-friendly dashboard interface
* ⚡ Fast and responsive (built with Vite + React)

---

## 🧠 Use Case

This system helps non-technical users access business insights without needing SQL or complex BI tools. It reduces dependency on data teams and enables instant decision-making.

---

## 🛠️ Tech Stack

* Frontend: React (Vite)
* Language: TypeScript / JavaScript
* Styling: CSS
* Data Handling: CSV parsing

---

## 📂 Project Structure

src/ → Main application code

package.json → Dependencies and scripts

## ⚙️ Installation & Setup

Clone the repository:
git clone [https://github.com/victorianelapati/analyze-purchase-ai-dashboard.git]

Navigate to project:
cd analyze-purchase-ai-dashboard

Install dependencies:
npm install

Run the project:
npm run dev
